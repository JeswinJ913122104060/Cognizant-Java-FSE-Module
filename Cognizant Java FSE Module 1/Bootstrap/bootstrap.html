2.1 & 2.2: Bootstrap Folder Structure & JavaScript Plugins

  • css/ - contains compiled CSS files.
  • js/ - contains JavaScript plugins and bundle.
  • icons/ - optional folder with SVG icon files.

JS plugins are included via bootstrap.bundle.min.js, which contains Popper for dropdowns, modals, tooltips, etc.
